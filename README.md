# hinet-speed-test

Download a file from Hinet speed test server to measure your network speed in the CLI.

Compile command:

> make

> ./speed_test -s [No of file size] -t [No of thread]

Available file size list:

test_010m.zip

test_015m.zip

test_020m.zip

test_040m.zip

test_060m.zip

test_100m.zip

test_200m.zip

test_250m.zip

test_400m.zip
